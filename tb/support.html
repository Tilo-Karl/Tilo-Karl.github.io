FAQ

How does it work?
We use Apple’s Call Directory to block Sweden’s telemarketing range 075-557 on-device.

Does it replace NIX?
No. NIX can fail; True Blocker automatically blocks 075-557 calls regardless.

Do you collect any data?
No tracking, no analytics, no servers. Everything is on-device.

Hur fungerar det?
Vi använder Apples Call Directory för att blockera svenska intervallet 075-557 direkt i enheten.

Ersätter appen NIX?
Nej. NIX kan fallera; True Blocker blockerar 075-557 automatiskt.

Samlar ni in data?
Ingen spårning, inga analyser, inga servrar. Allt sker i enheten.
